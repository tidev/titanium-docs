# Modules.Barcode

<TypeHeader/>

## Overview

### System Requirements

- Titanium SDK 5.5.1+
- Camera Permissions set in your tiapp.xml:

``` xml
<ti:app>
  <!-- ... -->
  <ios>
    <plist>
      <dict>
        <key>NSCameraUsageDescription</key>
        <string>We need permission to access your device camera.</string>
      </dict>
    </plist>
  </ios>
  <!-- ... -->
</ti:app>
```
