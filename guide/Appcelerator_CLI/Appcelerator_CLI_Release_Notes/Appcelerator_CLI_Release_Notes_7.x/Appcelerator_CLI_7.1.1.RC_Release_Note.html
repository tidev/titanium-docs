# Appcelerator CLI 7.1.1.RC - 22 August 2019

Appcelerator CLI 7.1.1.RC is a patch release that includes one improvement and several bug fixes.

As of this GA release, the previous CLI patch release is no longer supported. End of support for this version will be up to 2020-02-22 or until the next patch release. Note: major and minor releases continue to be supported according to their nominal lifetime. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.

# Improvements

  • ALOY-1697 (opens new window) - Add WebView "onlink" callback support

    • added support for setting onlink in views

  • CLI-1362 (opens new window) - Update to latest acs cli

    • Updated acs to version 2.1.2

# Fixed issues

  • ALOY-1698 (opens new window) - Compile: Errors thrown during sourcemap step are not visible to a user

  • CLI-1355 (opens new window) - Android "appc run" will always trigger rebuild if "tiapp.xml" is missing encryption policy property

  • CLI-1359 (opens new window) - When commenting out modules in the Tiapp.xml and running a build all modules move to one line

  • CLI-1363 (opens new window) - Daemon: Can't detect iOS 13 simulators

  • CLI-1365 (opens new window) - Daemon is not shut down when downloading a core via appc use

  • CLI-1366 (opens new window) - Studio restarts daemon during the appc use switching
