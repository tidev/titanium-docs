# Appcelerator CLI 8.1.1 GA - 15 September 2020

Appcelerator CLI 8.1.1 GA is a patch release that includes bug fixes.

# Component versions

The following components are shipped with CLI 8.1.1:

Alloy

1.15.2

Cloud CLI

2.1.8

Titanium CLI

5.2.4

Daemon

3.2.0

# Fixed issues

  • ALOY-1738 (opens new window) - Regression in behaviour of the Require XML tag

  • ALOY-1737 (opens new window) - Alloy: ListView dataBinding creates deprecated setItems call CLI-1394 (opens new window) - Duplicate symbol error when using Appc Source Code Encryption

    • Fixes a potential symbol clash between appc-verify and some modules, this fix is available for use in SDK 9.2.0 and above, and also SDK 9.1.0 and below

  • CLI-1396 (opens new window) - Always prompted for iOS code base on module creation even if set

  • CLI-1397 (opens new window) - Update appc-verify to be built as an xcframework

    • Adds SDK 9.2.0 compatability to appc-verify which is used for Appc Source Code Encryption

  • CLI-1398 (opens new window) - Update to Titanium CLI 5.2.4

    • This fixes an issue where symlinks were not preserved when installing Titanium
