# Titanium.UI.Windows

Windows Phone specific UI capabilities. All properties, methods and events in this namespace will only work on Windows or Windows Phone devices.

Availability
4.1.0
Extends
Titanium.Module

DEPRECATED SINCE 9.0.0

As of Titanium 9.0.0, building Windows apps is no longer supported.
