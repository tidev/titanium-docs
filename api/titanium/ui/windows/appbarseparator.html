# Titanium.UI.Windows.AppBarSeparator

Vertical bar to divide controls in a Windows application bar

Availability
4.1.0
Extends
Titanium.Module

DEPRECATED SINCE 9.0.0

As of Titanium 9.0.0, building Windows apps is no longer supported.

# Overview

To create an application bar separator, use the Titanium.UI.Windows.createAppBarSeparator method.

The separator only appears on Windows Store applications.
